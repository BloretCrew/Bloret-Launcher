## 功能与计划
- [ ] 下载 Minecraft
- [ ] 启动 Minecraft
- [x] Minecraft 数据查询
- [x] 日志
- [ ] 支持深浅色
- [ ] 百洛谷通行证登录
- [ ] 微软账户登录
- [ ] 离线登录
- [ ] ……

## 安装&使用
到[这里](https://github.com/BloretCrew/Bloret-Launcher/releases)下载最新版本的压缩文件，解压到合适位置后，打开 `Bloret-Launcher.exe` 即可。
